# VerdictLog

Every mod action gets a reason. Every appeal gets an answer.

VerdictLog is a Devvit Web moderation app that gives Reddit moderators appeal-ready decision memory.

## Core features

- Log Verdict from post and comment menu actions
- Search User Case File
- Copy Appeal Summary for modmail
- Configure Rule List
- Data Safety Controls with TTL and manual delete

## Built with

- Devvit Web
- TypeScript
- React
- Devvit server endpoints
- Redis
- Reddit API
- Menu actions
- Forms

## Scope rules

This app does not include:

- AI summaries
- analytics dashboards
- mod performance tracking
- punishment recommendations
- automatic escalation
- notifications
- multi-subreddit sync
