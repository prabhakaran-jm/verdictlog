# Product: VerdictLog

VerdictLog helps Reddit moderators remember why moderation actions were taken.

Positioning:
Reddit's mod log tells teams what happened. VerdictLog helps them remember why.

Tagline:
Every mod action gets a reason. Every appeal gets an answer.

Primary user:
Reddit moderators handling removals, warnings, bans, and appeals.

Core workflow:
1. Mod sees a post or comment that needs action.
2. Mod clicks Log Verdict.
3. App pre-fills username, content link, content type, and timestamp.
4. Mod selects rule, severity, and decision template.
5. Mod adds one short reason.
6. Later, another mod searches the username.
7. App shows the user's chronological case file.
8. Mod copies an appeal summary for modmail.

MVP only:
- Log Verdict
- Search User Case File
- Copy Appeal Summary
- Configure Rule List
- Data Safety Controls
