# Scope Guardrails

Do build:
- Simple moderator workflows
- Fast logging
- Fast search
- Appeal-ready summaries
- Minimal data storage
- Manual delete
- TTL-based retention

Do not build:
- AI features
- charts
- analytics
- mod scoring
- punishment recommendations
- notifications
- automatic escalation
- multi-subreddit sync
- external APIs

Every feature must support the demo:
Log why in 5 seconds.
Recall it in 2 seconds.
Answer appeals fairly.
